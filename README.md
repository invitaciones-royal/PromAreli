# Invitación Digital - Lic. Areli Gurrola

Web diseñada con estilo minimalista y elegante para conmemorar la graduación de la Lic. Areli Gurrola.

## Características
- Header con video de fondo responsivo.
- Galería dinámica de 49 fotografías.
- Reproductor de audio integrado.
- Diseño adaptado a móviles y escritorio.

## Instrucciones de carga
1. Sube el video a la carpeta `assets` con el nombre `video-header.mp4`.
2. Sube las fotos a `assets` nombradas del `img (1).jpg` al `img (49).jpg`.
3. Sube el archivo de audio como `musica.mp3` en la raíz.
4. Activa **GitHub Pages** en los ajustes del repositorio.
